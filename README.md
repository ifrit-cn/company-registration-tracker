# Ifrit Company Registration Tracker

A simple web-based tracker to monitor the registration progress of **Ifrit International Logistics Co., Ltd.** in Yiwu, China.

## What is this?

This website helps track:
- **Registration phases** - Current status of each registration stage
- **Document requirements** - What documents are needed and their status
- **Overall progress** - Visual progress indicators
